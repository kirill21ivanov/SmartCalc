Проект Smart_Calc_v1.0 by haywardm
Операции:
1. Скобки
2. Сложение (+)
3. Вычитание и унарный минус (-)
4. Умножение (*)
5. Деление (/)
6. Остаток от деления (%)
7. Возведение в степень (^)
8. Квадратный корень числа (sqrt)

Тригонометрические функции:
1. Синус (sin)
2. Косинус (cos)
3. Тангенс (tan)
4. Арксинус (asin)
5. Арккосинус (acos)
6. Арктангенс (atan)
7. Натуральный логарифм (log)
8. Десятичный логарифм (ln)

Тип сборки: qmake
Сборка проекта с помощью цели: make start
Удаление проекта с помощью цели: make uninstall
Тесты: make test
Покрытие тестами: make gcov_report
Описание проекта: make dvi
Калькулятор предназначен для рассчета арифметических операций и функций с возможностью построения графика точками.
Приложение также поддерживает функцию кредитного калькулятора.
